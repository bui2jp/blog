# vscode ショートカット

なかなか身につかないショートカットキー

## Mac のボタン

- ⌘ : Command
- ⌥ : Option/ALT
- ⇧ : Shift
- ^ : Control

## 編集

- 行の移動 : Alt + 上下
- 行のコピー : Shift + Alt + 上下
- 行の削除 : Shift + Command + K
- コード整形 : Shift + Alt + F

## 選択

- 短形選択 : Shift + Alt + ドラック
- 選択範囲の複数指定 : ⌘ + D 　（複数回押す）

## 移動

- コメント : Command + /

## パネル操作

- サイドバー : Command + B
- ターミナル : Shift + Ctrl + @
